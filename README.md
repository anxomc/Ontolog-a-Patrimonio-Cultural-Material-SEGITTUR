# Ontología de Turismo Cultural Material (Extensión de SEGITTUR)

[cite_start]Este repositorio contiene el trabajo práctico de la asignatura **Ingeniería Ontológica (2025-26)** [cite: 2] del Máster en Inteligencia Artificial.

[cite_start]El objetivo del proyecto es desarrollar una ontología OWL [cite: 3] [cite_start]que extiende la [ontología 'core' de SEGITTUR](https://ontologia.segittur.es/turismo/def/core/) [cite: 34] para modelar el caso de uso de **Turismo Cultural Material**.

[cite_start]El desarrollo sigue la **metodología NeOn** [cite: 3] [cite_start]y la implementación se realiza con herramientas como **CHOWLK** (para el modelado conceptual) y **Protégé** [cite: 3] (para la implementación OWL).

## Componentes del Proyecto
* **`/modelo-conceptual`**: Archivos `.drawio` del diagrama CHOWLK.
* **`/ontologia`**: Archivo `.ttl` de la ontología OWL implementada.
* [cite_start]**`/documentacion`**: Informe del proyecto, incluyendo la especificación de requisitos [cite: 10, 11] [cite_start]y la evaluación OOPS![cite: 27].
