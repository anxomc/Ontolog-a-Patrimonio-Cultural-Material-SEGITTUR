# Ontología de Turismo Cultural Material (Extensión de SEGITTUR)

Este repositorio contiene el trabajo práctico de la asignatura **Ingeniería Ontológica (2025-26)** del [Máster en Inteligencia Artificial](https://muia.dia.fi.upm.es/es/horarios/).

El objetivo del proyecto es desarrollar una ontología OWL que extiende la [ontología 'core' de SEGITTUR](https://ontologia.segittur.es/turismo/def/core/) para modelar el caso de uso de **Turismo Cultural Material**.

El desarrollo sigue la [**metodología NeOn**](https://oeg.fi.upm.es/index.php/es/methodologies/59-neon-methodology/index.html) y la implementación se realiza con herramientas como **CHOWLK** (para el modelado conceptual) y [**Protégé**](https://protege.stanford.edu/) (para la implementación OWL).

## Componentes del Proyecto
* **`/modelo-conceptual`**: Archivos `.drawio` del diagrama CHOWLK.
* **`/ontologia`**: Archivo `.ttl` de la ontología OWL implementada.
* **`/documentacion`**: Informe del proyecto, incluyendo la especificación de requisitos y la evaluación OOPS. Para visualizar la documentación de widoco en el formato correcto es necesario descargar la carpeta doc de la rama main y después abrir el archivo doc/widoco_documentation.html.
